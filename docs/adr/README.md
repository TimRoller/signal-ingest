# Architecture Decision Records

Each ADR is a short, durable explanation of one architectural choice — the context that forced the decision, the option chosen, and the consequences we accept by choosing it.

Format: [Michael Nygard's ADR template](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions).
